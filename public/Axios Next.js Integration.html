Here's how you can make the same request in Next.js using Axios:

Install Axios (if you haven't already)

Run this command in your Next.js project:

sh
npm install axios

Code:

javascript
import axios from "axios";

const apiKey = "YOUR_API_KEY"; // Replace with your actual API key
const baseUrl =
  "https://generativelanguage.googleapis.com/v1beta/models/gemini-pro:generateContent";

export const generateInterviewTips = async (skills, experiences, careerInterests) => {
  try {
    const url = `${baseUrl}?key=${apiKey}`;
    
    const response = await axios.post(
      url,
      {
        contents: [
          {
            parts: [
              {
                text: `Generate 40 interview tips in JSON format, each with "title" and "description" fields, based on skills: ${skills}, experiences: ${experiences}, and career interests: ${careerInterests}`,
              },
            ],
          },
        ],
      },
      {
        headers: { "Content-Type": "application/json" },
      }
    );

    return response.data; // Returns the API response
  } catch (error) {
    console.error("Error fetching interview tips:", error);
    return null;
  }
};

How to Use It:

If youre using this inside an API route in Next.js (/app/api/generate-tips/route.js for the App Router or /pages/api/generate-tips.js for the Pages Router), wrap it like this:

javascript
export default async function handler(req, res) {

      res.status(200).json(data);
    } else {
      res.status(500).json({ error: "Failed to fetch data" });

export async function POST(req) {

Key Fixes & Notes:

  1. File name:

    • Use route.js (not route.jsx) because this is an API route, not a React component.

  2. API Route Format:

    • In Next.js App Router, API routes use named functions like export async function POST(req).
    • req.json() is used instead of req.body to properly parse JSON data in the request.

  3. Response Handling:

    • Uses new Response() instead of res.status().json() because App Router API routes use Response objects instead of Express-style res.
